## 1. 프로젝트 개요

가족의 유럽 여행 준비를 돕고 기대감을 높이기 위한 **모바일 친화 정적 웹사이트**.
카테고리별 학습 본문을 읽고, 챕터 단위로 랜덤 퀴즈를 풀며 복습한다.

### 핵심 가치
- **프라이버시**: 서버·DB 없이 클라이언트 사이드에서만 동작. 가족의 개인정보를 외부로 보내지 않는다.
- **집중도 향상**: 제한 시간(10분)과 랜덤 문제 은행 방식으로 실제 시험 같은 몰입감을 제공한다.
- **확장성**: 카테고리·챕터·문제가 **코드 수정 없이** 데이터 파일만으로 계속 추가되는 구조.

## 2. 대상 사용자

- 함께 유럽 여행을 준비하는 가족 구성원 (부모님 포함, 다양한 연령대).
- 비개발자 가족도 콘텐츠를 쉽게 추가/수정할 수 있어야 한다 (JSON 편집 수준).

## 3. 정보 구조 (3계층)

```
카테고리 (Category)
  └─ 챕터 (Chapter)
       ├─ 학습 본문 (sections)
       └─ 퀴즈 문제 (questions)
```

- **카테고리**: 주제 대분류. 메인 화면에 목록으로 노출.
- **챕터**: 카테고리 내 학습 단위. 영어회화는 ~30챕터 규모를 상정.
- **본문/문제**: 챕터마다 함께 보유. 퀴즈는 챕터 단위로 출제.

### 화면 구성
- **🏠 메인 목차 (index)**: 전체 카테고리 리스트 제공. 카테고리 선택 시 챕터 목록으로 이동.
- **📋 챕터 목록 (category)**: 선택한 카테고리의 챕터를 스크롤 리스트로 제공.
- **📖 학습 본문 (chapter)**: 챕터 핵심 내용을 일목요연하게 제공. 하단에 `[퀴즈 시작하기]` 버튼.
- **📝 퀴즈 (quiz)**: 랜덤 10문항 + 상단 고정 타이머.
- **🎯 결과 (result)**: 점수 + 종합 해설.

## 4. 핵심 기능 요구사항

### 4.1 PIN 인증 및 퀴즈 시작
- `[퀴즈 시작하기]`를 누르면 PIN 입력 모달(Modal) 팝업을 띄운다.
- 클라이언트 사이드(JS 소스코드 내)에 지정된 PIN과 일치해야만 퀴즈 세션이 시작된다.
- PIN이 틀릴 경우 경고 메시지를 노출하고 진입을 차단한다.
- 퀴즈 화면 직접 진입(PIN 미통과) 시 본문 화면으로 되돌려 우회를 방지한다.
- **주의**: 클라이언트 사이드 특성상 PIN은 소스코드에 노출된다. 이는 실제 보안이 아니라 "장난 진입 방지용 가벼운 관문"이다. (README에 동일 명시)

### 4.2 세트 단위 랜덤 출제 (10문제 = 1세트)
- 문제는 **10문제짜리 세트** 단위로 구성한다. 한 챕터는 같은 개념을 다루되 구성이 다른 여러 세트(목표 10세트)를 가진다.
- PIN 인증 완료 즉시, 해당 챕터의 여러 세트 중 **무작위로 1세트(10문제)** 를 골라 세션을 생성한다.
- **암기 방지 강화:** 출제 시 세트 안 문제 순서와 각 문제의 보기 순서를 셔플한다.
- 세트가 1개뿐이면 항상 그 세트가, 세트 문제가 10개 미만이면 있는 만큼만 출제된다. 점수는 `맞힌 수 / 출제 수`로 표시.
- 문제는 한 페이지에서 세로 스크롤로 전 문항을 푸는 형태로 제공한다.
- **문제 유형:** 4지선다형(`multiple_choice`)과 O/X형(`ox`) 두 가지를 지원한다.

### 4.3 10분 제한 시간 타이머 및 자동 제출
- 퀴즈 시작과 동시에 **10분(600초)** 카운트다운 타이머가 화면 상단 고정 영역에 표시된다.
- 남은 시간이 60초 이하가 되면 타이머 색상을 강조(빨강)한다.
- 제한 시간이 만료되면 경고 창과 함께 **현재까지 선택한 답변을 기준으로 자동 제출**한다.

### 4.4 점수 화면 및 종합 풀이 제공
- 사용자가 직접 제출하거나 시간 만료로 자동 제출되면 결과 화면으로 전환된다.
- **최종 점수**: `맞힌 문항 수 / 출제 문항 수` 형태로 직관적으로 표시.
- **종합 해설 리스트**: 출제된 전 문항을 순서대로 나열하고, 사용자가 선택한 답, 정답 여부(O/X), 문제별 상세 풀이(`explanation`)를 한눈에 볼 수 있도록 제공.
- O는 초록, X는 빨강으로 정·오답을 시각 구분한다.
- `[다시 풀기]`(같은 챕터 새 랜덤) / `[목차로]` 버튼을 제공한다.

## 5. 비기능 요구사항

| 항목 | 요구사항 |
|------|----------|
| 플랫폼 | 모바일 우선, 데스크톱 호환 (반응형) |
| 의존성 | 외부 라이브러리·빌드 도구 없음 (순수 HTML/CSS/JS) |
| 오프라인 | 외부 폰트·CDN 미사용. 정적 자원만으로 동작 |
| 접근성 | 부모님도 보기 편한 큰 폰트·터치 타깃, 명확한 색 대비 |
| 데이터 저장 | 영구 저장 없음. 화면 간 전달은 sessionStorage(탭 닫으면 소멸) |

## 6. 데이터 스키마

### 6.1 `categories.json` (카테고리 메타 목록)
```json
[
  {
    "id": "finance",
    "icon": "💶",
    "title": "환전·결제 및 택스리펀",
    "summary": "트래블카드, DCC 차단, 택스리펀 절차",
    "file": "finance.json"
  }
]
```

### 6.2 `<category>.json` (카테고리 → 챕터 → 세트 → 문제)
```json
{
  "id": "finance",
  "title": "환전·결제 및 택스리펀",
  "chapters": [
    {
      "id": "ch01",
      "title": "현지 카드 결제 요령",
      "summary": "DCC 차단, 키오스크 결제",
      "sections": [
        { "heading": "DCC는 무조건 거절", "body": "단말기에서 KRW/EUR 선택이 뜨면 반드시 EUR을 고르세요." }
      ],
      "sets": [
        {
          "id": "set01",
          "questions": [
            {
              "id": "finance_ch01_set01_001",
              "type": "multiple_choice",
              "question": "현지 단말기에 KRW와 EUR 선택 창이 뜨면 무엇을 선택해야 할까요?",
              "options": ["원화 (KRW)", "현지통화 (EUR)", "아무거나 상관없음", "취소 후 현금"],
              "answerIndex": 1,
              "explanation": "KRW를 선택하면 이중 환전 수수료(DCC)가 발생합니다. 반드시 EUR을 선택하세요."
            },
            {
              "id": "finance_ch01_set01_002",
              "type": "ox",
              "question": "카드 단말기에서 원화(KRW) 결제를 선택하면 수수료를 아낄 수 있다.",
              "answerIndex": 1,
              "explanation": "아닙니다. 원화 결제(DCC)는 이중 수수료가 붙어 더 비쌉니다."
            }
          ]
        }
      ]
    }
  ]
}
```

**유형별 규칙**
- `multiple_choice`: `options`(보기 4개 권장, 최소 2개) + `answerIndex`(0-based).
- `ox`: `options` 없음. 화면에 O/X 두 버튼이 자동 표시되고, `answerIndex`는 **0 = O(참), 1 = X(거짓)**.
- 세트는 정확히 10문제로 구성한다. 문제 `id`는 `카테고리약어_챕터_세트_번호` 권장.

## 7. 기본 카테고리 (출시 시 5종)

| icon | 카테고리 | 내용 |
|------|----------|------|
| 🗣️ | 생존 영어 회화 | 공항·식당·길찾기 등 상황별 표현 (최대 ~30챕터) |
| 🏙️ | 방문 도시 상식 | 도시별 명소·교통·주의사항 |
| 🤝 | 문화·에티켓 | 팁 문화, 인사, 식사 예절 |
| 🎒 | 짐 싸기 체크리스트 | 소매치기 방지 용품(스프링 줄·다이얼 락), 멀티 어댑터, 비상약, 편한 운동화 |
| 💶 | 환전·결제 상식 | 트래블카드 사용법, 유로 환전 팁, DCC 차단, 공항 택스리펀 |

- 🚨 **위급 상황·안전 수칙**: 여권 분실 대처, 소매치기 빈번 구역·수법(싸인단, 오물 묻히기 등), 대사관 연락처, 긴급 전화번호. (동일 구조로 파일만 추가하면 됨)

## 8. 범위 밖 (Out of Scope)

- 사용자 계정/로그인, 서버·데이터베이스.
- 퀴즈 결과의 영구 저장 및 통계/리더보드.
- 다국어 UI (콘텐츠는 한국어 기준).
- 주관식·서술형 문제 유형 — 현재는 `multiple_choice`/`ox`만 지원, 향후 확장.
