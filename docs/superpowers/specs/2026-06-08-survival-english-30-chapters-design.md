# 생존 영어 회화 — 30챕터 구성 설계

작성일: 2026-06-08

## 목표

`survival-english`(생존 영어 회화) 카테고리를 **여행 전 과정을 시간순으로 따라가는 30챕터**로 확장한다.
대상은 **성인 초중급** 학습자(가족 유럽 여행)이며, 각 챕터는 **약 15분**에 학습할 수 있는 분량으로 나눈다.

기존 데이터 구조(`chapters[].sections` + `chapters[].sets`)와 10분 퀴즈 타이머를
그대로 사용하며, 코드 수정 없이 `data/survival-english.json` 만 채워 확장한다.

## 챕터당 분량 기준 (≈15분)

- **학습 본문**: 짧은 섹션 2~4개. 각 섹션은 `heading`(상황/주제) + `body`(핵심 표현 + 한글 설명). 읽기 약 5분.
- **퀴즈**: 10문항 세트 1~2개. `multiple_choice`와 `ox` 유형을 혼합. 10분 타이머 → 약 10분.
- 한 챕터 = **하나의 상황**. 외워서 바로 쓸 수 있는 핵심 표현 6~10개 수준.

## 데이터 구조 규칙 (기존 ch01 기준)

```
chapters[]
  id        "chNN" (2자리 0패딩, 시간순)
  title     상황명 (한글)
  summary   한 줄 요약
  sections[]  { heading, body }
  sets[]      { id "setNN", questions[] }
    questions[]
      id            "se_chNN_setNN_001" 형식
      type          "multiple_choice" | "ox"
      question      한글 질문
      options       (multiple_choice일 때) 보기 4개
      answerIndex   정답 인덱스 (ox는 0=O, 1=X)
      explanation   한글 해설
```

## 기존 챕터 재배치

| 기존 | 신규 위치 | 비고 |
|------|-----------|------|
| ch01 공항 입국 심사 | 8번 | 본문·퀴즈 완성됨, id를 ch08로 재배치 |
| ch02 식당에서 주문하기 | 19번 | id를 ch19로 재배치 |
| ch03 길찾기와 교통 (준비 중) | 15번 | "길 묻기"로 정리, id를 ch15로 |

> 챕터 id를 시간순으로 재배치하므로, 카테고리 화면의 챕터 정렬은 JSON 배열 순서를 따른다(파일 내 시간순 정렬 유지).
